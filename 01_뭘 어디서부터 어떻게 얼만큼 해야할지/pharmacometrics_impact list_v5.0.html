계량약리학이라는 무기로
"한 명이라도 더" 살린다

계량약리학의 본래 사명은 신약 개발과 임상 현장에서 '분자 이후에 벌어지는 실패'로 인해 죽거나 불필요하게 고통받는 사람 수를 한 명이라도 더 줄이는 것이다. 그 실패는 세 지점에서 발생한다 — 잘못된 용량 결정(Dose), 배제된 집단과 생존 위기 현장의 번역 실패(Population·Crisis), 새로운 모달리티에 맞지 않는 낡은 방법론(Modality)에서.

규범적 앵커 — 카라페탼·쉰들러 원칙
"이 차를 팔았다면 10명을 더, 이 배지를 팔았다면 2명을 더... 내가 돈을 조금만 더 벌었더라면 한 명이라도 더 살릴 수 있었을 텐데..." — 이 고백이 계량약리학이 짊어져야 할 윤리적 나침반이다.
Pillar I — 용량 결정 실패
Dose Selection Failure
MTD 패러다임이 표적치료제·ADC·이중항체·CAR-T에서 체계적으로 잘못된 용량을 허가받게 만들었다. Project Optimus + TGI-OS가 이 실패를 수리적으로 해결한다.
Pillar II — 배제·생존 위기 현장
Translation Gap & Critical Care Crisis
소아·임산부·CKD뿐 아니라 ECMO/CRRT 중환자와 ARC 소아 FN 환자가 지금 이 순간 표준 용량이 맞지 않아 죽어가고 있다. 8년 TDM 경험의 최대 발화점.
Pillar III — 모달리티 전환
Novel Modality Gap
ADC 4분석물, 이중항체 CRS-QSP, CAR-T DDE, mRNA-LNP Minimal PBPK — 신모달리티에 기존 방법론은 작동하지 않는다. 방법론 공백과 산업 수요가 일치하는 유일한 지점.
세 기둥의 교집합 — 한국 PhD 비교우위 좌표 (v5.0 · 4개)
좌표 1: ADC·이중항체 + Project Optimus + 한국 ADC 산업(LigaChem·Samsung Biologics·ABL·Celltrion)
좌표 2 ★신규: ECMO/CRRT·ARC 소아 FN MIPD + 한국 3차 병원 TDM 데이터 직접 접근
좌표 3 ★신규: TGI-OS 모델링 + Seoul 7년 연속 세계 최다 산업 주도 임상시험 도시
좌표 4: Agentic QSP-Copilot 응용 + 8년 임상 차트 리터러시 × LLM 워크플로우

5대 Reality Check — 실행 진흙탕의 전면 공개
⚠
Reality Check 01
ADC 데이터 수급의 구조적 불가능성
문제: LigaChem·Samsung Biologics의 ADC multi-analyte 원시 PK 데이터는 1년차 박사과정생에게 공개될 확률 0. AIMS 컨설팅 ADC 과제를 기다리는 것은 기도다. 출판된 농도 데이터의 85%가 원본과 5% 이상 불일치, QSP/PBPK 논문 83–94%가 이 사용 사실 자체를 미공개[확인 필요].
▶ Plan B — 역설계 시뮬레이션 코호트 + 추출 오류 민감도 분석 (v5.0 강화)
Enhertu NDA 761139 등 승인 ADC의 공개 PK 그래프를 WebPlotDigitizer로 역추출 + 추출 오류 ±10% 민감도 분석 의무화. 이 민감도 분석 자체가 CPT:PSP 투고 가능한 방법론 서명 논문. 실데이터 접근 시 Plan A 검증 논문으로 전환.
⚠
Reality Check 02
MIPD 3차 병원 구현의 속도계 오류
문제: IRB 심의(3–6개월) + 정보보호심의위(6–12개월) + EMR 연동 개발(6–18개월) + 파일럿(6–12개월). 낙관적으로도 2년, 박사 36개월 내 완주 불가. Swartling 2024(Uppsala) vancomycin MIPD 구현 논문이 현실적 경로를 실증.
▶ Plan B — 후향 코호트(축1) + 단일 병동 태블릿 관찰(축2) 이원화
EMR 연동·전향 파일럿 명시적 포기. 축1: 반코마이신·타크로리무스 TDM 3–5년치 후향 코호트 → 예측 정확도 검증. 축2: 단일 ICU 병동 오프라인 태블릿 Shiny 앱 관찰 연구 → IRB 장벽 최소화.
⚠
Reality Check 03
인지적 과부하 — SciML 타임박싱 원칙 (협상 불가)
문제: NONMEM + R + PK-Sim + 대학원 수업 + AIMS + 1st paper 위에 Julia 생태계(DifferentialEquations.jl · Zygote · DeepNLME)를 얹으면 번아웃이 구조적으로 보장된다.
▶ Pinned Hobby 모델 — 하드 원칙 2개
원칙 1: #1 ADC FOCE-I 수렴 성공 전까지 Julia REPL 열지 않는다. 원칙 2: T+30~36 박사 마무리 6개월간 SciML 완전 중단. 이 원칙 위반 시 #1·#3·#4까지 도미노 붕괴.
⚠
Reality Check 04 ★신규
후향 MIPD 코호트의 반사실 추론 결함
문제: 후향 TDM 데이터는 MIPD를 적용받지 않은 환자들의 데이터다. "만약 MIPD를 썼다면 어땠을까"를 이 데이터로 평가하는 것은 반사실적 추론(counterfactual reasoning)의 근본 문제를 내포한다.
▶ Target Trial Emulation 적용
후향 코호트 분석 범위를 "MIPD 알고리즘의 예측 정확도 검증"으로만 제한. "임상 결과 개선 인과 추론"은 명시적 보류. Target Trial Emulation 방법론으로 반사실 추론 한계 구조적 통제.
⚠
Reality Check 05 ★신규
Agentic AI 환각과 구조적 제어
문제: LLM은 단순 NONMEM 구획 모델 코딩에서 만점에 가깝지만, 복잡 다중 구획 모델·잔차 오류 모델링에서는 그럴듯한 환각 코드를 생성한다. 단일 챗봇에 의존하면 임상적 재앙이 발생할 수 있다.
▶ Multi-agent DAG + Human-in-the-Loop (Saini & Farnoud 2025 QSP-Copilot 아키텍처)
단일 챗봇 의존 절대 금지. DAG 기반 모듈형 multi-agent 워크플로우(문헌 추출 에이전트 → 지식 그래프 에이전트 → ODE 변환 에이전트 → 검증 에이전트) + 각 단계 전문가 감독 checkpoint. QSP-Copilot은 모델 개발 시간 약 40% 단축 실증[확인 필요].
