Sec · A 통계 방법론 — 결정과 근거
A.1 Gwet's AC2가 Cohen's κ보다 우월한가 Strong (조건부)

결론: 4범주 순위 코딩에서 Gwet's AC2(quadratic ordinal weights) 사용은 강하게 지지됩니다. 단, 단독 사용이 아닌 robustness panel의 primary로만 위치시킵니다.

Kappa Paradox 기전 (Feinstein & Cicchetti 1990 / Vach & Gerke 2023)

κ = (Po − Pe)/(1 − Pe). 한 범주 주변확률 → 1 시 Pe → 1, 분모가 분자보다 빠르게 0으로 수렴 → 관찰 일치가 높아도 κ가 인위적으로 deflate. "silent" 또는 "recommended"가 우세한 본 연구는 Paradox 1 고위험 상황에 해당.

A.3 Sample Size / Power 계산 Moderate
Worked Example — kappaSize::CI4Cats()
kappaSize::CI4Cats(
  kappa0 = 0.70,        # 기대 AC2
  kappaL = 0.60,        # 목표 95% CI 하한
  kappaU = 0.80,        # 목표 95% CI 상한
  props  = c(0.25, 0.35, 0.10, 0.30),  # 파일럿 후 보정
  raters = 3,
  alpha  = 0.05
)
# 필요 item 수 ≈ 55–80
# 실제 80 item × 3기관 = 240 → 목표 precision 충분히 만족

주의: kappaSize는 intraclass κ 전용. AC2 전용 표준 R 도구 미존재 → "similar large-sample variance" 근사임을 methods에 명시 필수. 도메인별 stratification(도메인당 ~20 item)은 CI 반폭 0.18–0.22로 넓어지므로 pooled AC2 primary, 도메인별은 탐색적 caveat와 함께 보고.

D.1 Sino-US-DrugQA 벤치마크 검증 Moderate
Preprint, NOT peer-reviewedUS-China onlymedRxiv 2026.02

Fu, Chen, Lu. medRxiv 2026.02.13.26346236. 11,871 MCQ, US FDA / CFR Title 21 vs China NMPA. 최고 모델 Gemini-3-flash (κ=0.764). Overall zero-shot 78.97–84.51%, cross-jurisdictional comparative에서 6–9pp drop. 수치는 정확하나 MFDS/Korea 데이터 없음.

권장 인용 문구: "As analogical evidence, the recent Sino-US-DrugQA benchmark (Fu et al., medRxiv preprint, February 2026) demonstrated that frontier LLMs' monolingual retrieval accuracy (78.97–84.51%) drops 6–9 percentage points in cross-jurisdictional comparative reasoning tasks. Although that benchmark did not include Korean or MFDS content, the pattern motivates our restriction of LLM use to extraction and the reservation of comparative coding to human experts."

# ROLE
You are a regulatory science extraction assistant. Your only job is
to return verbatim text from the provided document with precise
source metadata. You MUST NOT paraphrase, summarize, or synthesize.

# SETTINGS (API call)
temperature = 0
max_tokens = 8192
response_format = JSON (if supported)

# TASK
From {DOC_TITLE} ({AGENCY: FDA | EMA | MFDS}), extract every passage
relevant to {TOPIC, e.g., population pharmacokinetic analysis
requirements for antibody-drug conjugates}.

# RULES
1. EXTRACT ONLY. Exact source text. For Korean sources, reproduce
   verbatim in `original_text` (language="ko-KR"); provide literal
   English in `english_literal` with translation_method="literal_non_official".
2. Each passage: document_id, agency, version/date, page,
   section_number, section_heading, paragraph_index, original_text,
   english_literal, relevance_rationale (≤25 words), confidence
   {HIGH,MEDIUM,LOW}, uncertainty_flags.
3. Korean→English: preserve regulatory terms untranslated on first
   mention with bracketed literal gloss, e.g.,
   "임상시험계획서 [clinical trial protocol]".
4. Output ONLY valid JSON. No markdown, no commentary outside JSON.
5. requires_human_review = true if: confidence=LOW,
   korean_ambiguous_term, crosses section boundary,
   contains numeric requirement (dose, sample size, %, N).
