Chapter 2.5 · Clearance Concepts

소실청소율(CL): 약동학의 심장

CL은 단순한 숫자가 아니라 — 간·신장·혈류가 약물을 어떻게 "처리"하는지의 총합적 생리학적 역량이다.

1.1 정의와 AUC 역관계

기본 정의 — Eq 2:180 dX/dt = CL · Cp   // 소실속도 = CL × 혈장농도 CL(IV) = D_iv / AUC_iv(0→∞) CL/F(PO) = D_po / AUC_po  // CL↑ → AUC↓ (반비례)

⚠️ 초보자의 함정 — CL ≠ 소실속도

CL ≠ 소실속도(elimination rate). 소실속도는 CL × Cp이므로 농도에 따라 변하지만, CL 자체는 선형 PK에서 농도와 무관한 고정 파라미터다.

1.2 추출비(ER)와 Well-Stirred Model

Well-Stirred Model — Eq 2:186–2:195 E_H = (C_in − C_out) / C_in Cl_H = Q_H · f_u · Cl_int / (Q_H + f_u · Cl_int) F_H = 1 − E_H = Q_H / (Q_H + f_u · Cl_int)

저추출 약물: 간경변 → 알부민↓ → f_u↑, 동시에 Cl_int↓. 두 효과가 상쇄되어 총 CL이 별로 변하지 않는 것처럼 보일 수 있다. 실제로는 약물마다 다르므로 비결합 Cl_u = f_u·Cl_int를 별도로 추적해야 한다.

PO 정상상태 농도에서 Q_H/(Q_H + f_u·Cl_int) 항이 분자·분모에서 상쇄된다. 결과적으로 고추출이든 저추출이든 f_u·Cl_int로 수렴 — 이것이 CL/F에서 F를 분리할 수 없는 수학적 이유다.

Why-1 간경변 + Propranolol: 생체이용률 3배 증가 메커니즘 ▼
🩺 임상 상황

Propranolol (ER ≈ 0.7, 고추출). 정상인: F ≈ 25%. 간경변 환자: 동일 경구 용량에서 AUC 3배↑, F ≈ 75%. 동일한 약, 왜?

메커니즘 1: 간혈류 감소

고추출에서 F_H = Q_H/(Q_H + f_u·Cl_int). 간경변 → Q_H↓ → F_H = 1−E_H↑. 혈류가 느릴수록 처리 시간 없이 통과.

메커니즘 2: 문맥-전신 단락 (Porto-systemic Shunting)

문맥압↑ → 간을 우회하는 단락 형성. 경구 흡수된 약의 상당량이 first-pass를 완전 우회. 혈류 감소보다 훨씬 강력한 효과.

메커니즘 3: Cl_int 감소

간세포 수↓, 효소 활성↓ → f_u·Cl_int↓ → E_H↓ → F_H↑. 세 메커니즘이 모두 같은 방향.

⚠️ 저추출 약물(Warfarin)은 다르다

저추출에서 F_H는 처음부터 ≈1. 간혈류 감소가 F에 영향 없다. 대신 Cl_int↓ → CL↓ → t½ 연장이 주 문제.

🏛️ 규제 시사점

Child-Pugh B/C 고추출 경구 약물: AUC 증가 3~5배 예상. FDA Hepatic Impairment Guidance는 F 변화를 별도로 요구.

Why-2 경구 데이터만으로 절대 CL을 보고할 수 없는 수학적 이유 ▼

결론: F와 CL은 경구 데이터만으로는 수학적으로 분리 불가능(structurally unidentifiable)하다.

비식별 관계 CL/F = D_po / AUC_po  // 추정 가능 CL = F × (CL/F)    // F를 모르면 CL도 모른다 CL=10, F=0.5 ≡ CL=5, F=1 → 동일한 AUC 생성

🏛️ FDA 함의

PopPK 보고서에서 PO 기반 CL/F를 "CL"로 기재하면 심사관이 즉시 지적한다. DDI 예측, PBPK 입력값에는 반드시 IV 파트가 필요하다.

Why-3 FDA 신기능 질의 방어 — eGFR 공변량으로 CL/F를 지키는 법 ▼
🏛️ FDA 가상 질의

"신장·간 CL을 분리하지 않은 상태에서 신장애 하위군 예측을 어떻게 보장할 것인가?"

전략 1: eGFR 공변량 모델링

CL/F_i = θ_CL × (eGFR_i / eGFR_median)^θ_eGFR × exp(η_CL,i) // θ_eGFR 추정값과 95%CI로 신기능 의존도 정량화

전략 2: 시뮬레이션 기반 용량 정당화

eGFR 15/30/60 mL/min 집단 가상 시뮬레이션 → AUC_ss 분포 비교 → 노출 차이 정량화.

전략 3: Stratified VPC

신기능 그룹별 VPC 생성 → 모든 하위군에서 90% 피복률 시각적 증명.

💡 핵심

FDA는 "기관별 CL 분리"보다 "신기능 하위군 예측 정확도"를 더 중시한다. 공변량 + VPC + 시뮬레이션의 삼각 구성이 방어의 황금률.

Chapter 2.6 · Target-Mediated Drug Disposition

TMDD: 항체가 표적에 결합하면 PK가 바뀐다

항체의약품 시대에 TMDD를 모르면 임상 PK 해석에서 반드시 틀린다. "왜 초기엔 t½이 짧다가 반복 투여 후 길어지는가?" — 이 질문의 답이 TMDD다.

2.1 생물학적 본질 — 수용체가 소각로다

소분자 약물은 수용체에 결합해도 복합체 자체가 제거 경로가 되지 않는다. 그러나 mAb는 표적 결합 후 수용체 매개 내재화로 리소솜에서 분해 → 농도 의존적 추가 제거 경로를 만든다.

⚠️ 식별성 함정 — 파라미터가 너무 많다

Full TMDD는 9+개 파라미터. 희소 임상 PK 데이터로는 k_on, k_off를 개별 추정 불가. 이 둘은 오직 K_SS = (k_off+k_int)/k_on의 조합으로만 데이터에 나타난다. NONMEM 실행 시 %RSE > 100% 또는 수렴 실패가 일반적.

🩺 임상 상황

허셉틴(HER2 표적). 1차 투여 후 t½ ≈ 1.4일 → 4주 반복 후 정상상태 t½ ≈ 28일. 동일 약·동일 환자에서 왜?

HER2 표적 포화에 따른 유효 제거속도 변화

초기 투여 (포화 미달)

k_eff ≈ k_el + k_int · R / (K_SS + L) // L << K_SS → k_eff ≈ k_el + k_int·R₀/K_SS (빠름)

HER2 과발현 → 유리 수용체 R₀ 풍부 → k_int 경로 최대 가동 → 짧은 t½.

반복 투여 후 정상상태 (포화)

누적 항체가 HER2 수용체를 포화 → 유리 R → 0 → k_int 경로 꺼짐 → FcRn 경로(k_el)만 남음 → t½ ≈ 28일.

⚠️ 흔한 오해

"반복 투여 t½ 증가 = ADA 없어서 축적?" — 아니다. ADA는 t½을 단축시킨다. 여기서 t½ 증가는 순수 TMDD 포화 메커니즘이다.

모델 COU가 "농도 예측·용량 추천"이면 QSS로 충분하다. FDA도 항상 가장 복잡한 모델이 아닌, COU에 맞는 모델을 원한다.

Why-3 BLA 제출 시 "Full 대신 QSS" — FDA 방어 5-layer 논리 ▼
🏛️ FDA 가상 질의

"Full TMDD 대신 QSS를 채택한 과학적 근거는? QSS 가정의 타당성을 어떻게 검증했는가?"

BLA 5-Layer 방어 구조

논거 1: 데이터 지지

Full TMDD: k_on %RSE=178%, k_off %RSE=203% QSS: K_SS %RSE=14%, k_int %RSE=22%

논거 2: 수렴 안정성

Bootstrap 1,000회: Full TMDD 수렴 실패 43%, QSS 2%.

논거 3: VPC 동등성

Stratified VPC: Full 91% vs QSS 93% 피복률 — 유의한 차이 없음.

논거 4: 생물물리학적 근거

mAb 결합속도(k_on ~ 10⁵–10⁶ M⁻¹s⁻¹)는 PK 타임스케일(시간–일)보다 수십만 배 빠름 → QSS 가정 타당.

논거 5: COU 적합성

모델 COU = 노출 특성화·용량 추천. 수용체 점유율 절대값 필요 시 Full TMDD를 별도 PD 분석에 사용.

💡 방어의 황금 구조

데이터 지지 → 수렴 안정 → VPC 동등 → 문헌 근거 → COU 적합
