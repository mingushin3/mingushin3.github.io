미카엘리스-멘텐(MM) 역학 — 핵심 방정식과 직관

MM ODE (Eq 2:266–2:272) dC/dt = −Vmax·C / ((Km + C)·V) Cl(C) = Vmax / (Km + C)  // 농도에 따라 변함! AUC₀∞ = C⁰/Vmax · (Km + C⁰/2) t½(순간) = ln2·Vss/Vmax·(Km+C)
🩺 임상 직관 — 좌석 비유

CYP 효소 = 좌석. Km = 절반이 찼을 때 약물 농도. C << Km이면 좌석 여유 → 1차 선형. C >> Km이면 좌석 포화 → 0차(Vmax) 일정 속도.

Phenytoin TDM: 혈중 농도가 Km(≈5–15 μg/mL) 근처 → 치료 농도 = 비선형 위험 구간이 완전히 일치!

Why-1 Phenytoin 증량 함정 — Css = Km·D/(Vmax−D)의 수학적 의미 ▼
💡 핵심 공식 도출

정상상태에서 소실속도 = 투입속도: Vmax·Css/(Km+Css) = R₀

정리 → Css = Km·R₀/(Vmax − R₀)

분모 (Vmax − R₀)가 0에 수렴하면 Css는 ∞로 발산. 이것이 Phenytoin의 임상 함정이다.

수치 계산 (Vmax=533, Km=11.7) 200 mg: Css = 11.7×200/(533−200) = 7.0 μg/mL (치료 미달) 300 mg: Css = 11.7×300/(533−300) = 15.1 μg/mL ✓ 400 mg: Css = 11.7×400/(533−400) = 35.2 μg/mL ✗ 독성! 480 mg: Css = 11.7×480/(533−480) = 106 μg/mL ✗✗ 치명적
⚠️ NONMEM 코딩 함정

ADVAN10($PK에서 VMAX, KM 정의)에서 KM 단위는 농도(μg/mL)가 아닌 amount(mg)다. $DES(ADVAN13)는 농도 단위. 이 단위 불일치는 침묵 오류의 주범이다.

전략 1: Dose-normalized AUC 통계 검정 (Power model)

AUC = α · Dose^β // β=1: 선형, β>1: MM 포화, β<1: 피드백/유도 H₀: β=1 (선형성) → t-검정 또는 Likelihood ratio test

전략 2: 기전 근거 제시

In vitro 데이터(마이크로솜 Vmax, Km) → In vivo 임상 AUC 패턴과 일치성 비교. 포화 효소 동정(예: CYP2C9).

전략 3: 특수 집단 시뮬레이션

신장애/간장애에서 Vmax·Km 변화를 가정한 시나리오 시뮬레이션 → AUC 분포 제시.

💡 설계 핵심

Phase I에서 비선형성이 의심되면 Km/10 ~ Km×10 농도 범위를 커버하는 용량 설계가 필요하다.

시간의존 비선형 — 자가/이종 유도 (Autoinduction/Heteroinduction)

💡 핵심 메커니즘 (Eq 2:275–2:288)

효소가 약물에 의해 유도되면 Vmax가 시간에 따라 증가한다. 결과: 반복 투여 시 Css가 점점 감소하고 t½이 단축된다.

E(t) = E'ss − (E'ss − E₀)·e^(−kout·t)

kout = 효소 분해 속도 상수 (= 유도 t½을 결정)

🩺 Carbamazepine 임상 앵커

투여 초기 Css 높음 → 2~4주 후 Css가 절반으로 감소 → "내성 발생?" 아니다, 자가유도(autoinduction)로 Vmax가 증가했기 때문이다.

단백결합 비선형 — in vitro 폐쇄계 vs in vivo 개방계 함정

⚠️ in vitro 폐쇄계 오해 (Fig 2.104)

알부민 추가 → 유리약물↓ → 효과↓. "단백결합 높을수록 효과 감소" — 폐쇄계 착각이다.

실제 임상에서 알부민 증가는 효과를 변화시키지 않는다.

✅ in vivo 개방계 진실 (Fig 2.105)

Cu = K₀/Clu — 알부민이 늘어도 K₀(투입속도)와 Clu(비결합 CL)가 동일하면 Cu는 변하지 않는다.

결론: 단백결합 변화만으로는 임상 효과가 바뀌지 않는다.

🏛️ 규제 함의 — Warfarin + Phenylbutazone DDI의 진실

두 약물 모두 단백결합 높음(fu 0.01, 0.03). 상호작용이 단백결합 치환 때문이라고 교과서에 기재됐었다. 실제 원인: CYP2C9 공유 대사 억제.

Chapter 2.8 · Non-Compartmental Analysis

비구획분석 (NCA): 모델 없는 PK 파라미터 추출

NCA는 구획 모델 가정 없이 실측 데이터에서 직접 PK 파라미터를 계산한다. Phase I 분석의 출발점이자, 규제 제출의 기본 데이터다.

NCA (비구획분석)

모델 없이 사다리꼴 면적으로 AUC 계산. 선형(1차) 소실 가정만 필요. 자동화 용이.

AIMS Phase I 첫 번째 분석

모든 코호트 완료 즉시 NCA 실행 → 용량 비례성 확인 → 다음 코호트 용량 결정.

회귀분석 (모델 기반)

명시적 구획 모델(Ka, K, V) 가정. 개체 내 변이 분리 가능. 모델 선택이 결과에 영향.

주의

모델 선택이 틀리면 파라미터가 틀린다. NCA는 이 위험이 없다 — 그것이 NCA의 핵심 장점이다.

Why-1 Phase I에서 NCA가 첫 번째 PK 분석인 이유 ▼

Phase I에서는 어떤 구획 모델이 적합한지, 비선형성이 있는지, 흡수 패턴이 어떤지 알 수 없다. NCA는 이 모든 가정 없이 순수하게 데이터에서 파라미터를 추출한다.

NCA 핵심 파라미터 Cl = D_iv / AUC₀∞ Cl/F = D_po / AUC₀∞ Vss = MRT · Cl t½ = ln2 / λz F = (AUCpo/Dpo)/(AUCiv/Div)
💡 규제 역할

FDA IND 제출: 모든 Phase I 코호트의 NCA 요약표 필수. CDISC ADNCA 포맷. NCA Cmax/AUC 결과가 dose escalation 기준의 근거가 된다.

Why-3 FDA 대응 — "외삽 비율 20% 초과 — 신뢰할 수 있는가?" (Eq 2:324) ▼
🏛️ FDA 가상 질의

"AUC₀∞의 26%가 Clast/λz 외삽에서 왔다. 외삽 기반 AUC를 신뢰할 수 있는가?"

외삽 비율 (Eq 2:324) % 외삽 = AUC(tlast→∞)/AUC(0→∞) × 100 // 교재 권고: <20–25%. 초과 시 신뢰도 의문

4-layer 방어 전략

1. R²adj ≥ 0.98 제시: 말단 상 회귀 적합도 증명 → λz 추정 신뢰성 확보.

2. 채혈 설계 사유 설명: 반감기 예측 어려움, 환자 부담 등 실용적 이유 명시.

3. 구획 모델링과 비교: NCA Cl vs 회귀분석 Cl 일치성 제시.

4. 향후 연구 계획: Phase II에서 채혈 시점 연장 계획 명시.

💡 AUMC >> AUC 외삽 민감도

AUC₀∞는 외삽 비율 25%여도 견딜 수 있다. 그러나 AUMC₀∞는 외삽에 훨씬 더 민감하다. MRT·Vss 계산 시 외삽 오차가 2차 증폭된다. 외삽 >20%라면 NCA Vss 대신 구획 모델링 Vss를 우선한다.
